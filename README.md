# Система управления проектами

Корпоративная веб-система для управления проектами и задачами.

## Технологии

- **Backend:** Java 17, Spring Boot 3
- **Frontend:** Thymeleaf, Bootstrap 5
- **Database:** PostgreSQL 15
- **Migration:** Liquibase
- **Build:** Maven

## Быстрый старт

### Требования

- Java 17+
- PostgreSQL 15+
- Maven 3.8+

### Установка

1. Клонировать репозиторий
2. Создать базу данных `project_system`
3. Настроить `application.properties`
4. Запустить: `mvn spring-boot:run`

### Доступ

- URL: http://localhost:8080
- Логин: admin
- Пароль: admin
